FAQ

Ho bisogno di Tor o I2P per usare questa app?

No. La Clearnet funziona da sola e ti dà accesso a oltre 50.000 stazioni tramite l'API RadioBrowser, come una normale app radio. Tor e I2P sono completamente opzionali per gli utenti che desiderano privacy o accesso alle stazioni darknet.

Come posso rimanere privato/anonimo usando questa app?

Per impostazione predefinita, l'app si connette direttamente dal tuo indirizzo IP. Per rimanere anonimo, devi abilitare una modalità forzata nelle impostazioni e/o eseguire una VPN in background. Ci sono diverse opzioni a seconda della tua configurazione:

  • Forza Tor per tutto — Instrada tutto il traffico (stream, chiamate API, copertine) attraverso Tor. Nulla tocca Internet direttamente. Se Tor si disconnette, tutto il traffico viene bloccato — non ci sono fallback clearnet.
  • Forza Tor eccetto I2P — Instrada il traffico clearnet attraverso Tor, ma invia il traffico I2P attraverso il proxy HTTP I2P. Questo evita la latenza inutile dell'instradamento di I2P su Tor.
  • Forza proxy personalizzato — Instrada tutto il traffico attraverso un proxy che configuri tu (SOCKS4, SOCKS5, HTTP o HTTPS). Utile se gestisci il tuo proxy o usi un provider proxy non-Tor.
  • Forza proxy personalizzato eccetto Tor/I2P — Instrada il traffico clearnet attraverso il tuo proxy personalizzato, mentre le stazioni Tor e I2P usano i loro proxy nativi.

Per la massima privacy: abilita una modalità forzata, disabilita l'API RadioBrowser se non hai bisogno delle stazioni clearnet, disabilita il caricamento delle copertine per eliminare tutte le richieste di immagini, abilita la crittografia del database per proteggere i tuoi dati locali a riposo, e esegui una VPN in background insieme alla modalità proxy di InviZible Pro per un ulteriore livello di protezione se non ti fidi completamente dell'app o delle API a cui si connette.

Tutte le modalità forzate sono a prova di errore — se il proxy va giù, il traffico viene bloccato, non esposto. Questo include la prevenzione delle perdite DNS — nelle modalità forzate, tutte le query DNS vengono risolte attraverso il proxy anziché dal DNS del tuo sistema/ISP.

Se non ti fidi di questa app, esegui una VPN affidabile in background.

Quali dati raccoglie questa app?

Nessuno. Tutti i dati (password proxy, stazioni, cronologia, impostazioni, preferiti) sono memorizzati localmente sul tuo dispositivo. Non c'è sincronizzazione cloud, nessuna analisi e nessuna telemetria. Puoi opzionalmente crittografare tutti i dati locali con SQLCipher.

Le stazioni clearnet bloccano Tor nella modalità Tor forzata?

A volte, ma raramente. La stragrande maggioranza delle stazioni clearnet non blocca Tor.

Le copertine si caricano attraverso il mio proxy?

Sì. Le copertine delle stazioni vengono caricate attraverso la stessa pipeline proxy degli stream su tutte e tre le reti (clearnet, I2P e Tor). Nelle modalità forzate, tutte le richieste di immagini includono la protezione completa contro le perdite DNS — il DNS viene risolto attraverso il proxy, non dal resolver di sistema. Se il proxy si disconnette, il caricamento delle immagini viene bloccato anziché tornare a una connessione diretta. Puoi anche disabilitare completamente le copertine nelle impostazioni per eliminare tutte le richieste di immagini.

Abilitare Tor (modalità non forzata) nasconde il mio IP?

Non necessariamente. Senza abilitare una modalità forzata, tutto il traffico clearnet (stream, copertine) verrà caricato dal tuo IP diretto. Abilitare questa modalità non forzata ti permette semplicemente di accedere facilmente alle stazioni Tor senza dover instradare tutto attraverso Tor.

Come posso importare stazioni da un'altra app?

Impostazioni → Importa. Formati supportati: CSV, JSON, M3U, PLS.

Perché il traffico in chiaro è abilitato nel manifest?

I siti I2P (.i2p) e Tor (.onion) usano http:// anziché https://. Il traffico in chiaro deve essere abilitato affinché questi domini funzionino.

Posso usare un proxy diverso da InviZible Pro?

Sì. Qualsiasi proxy SOCKS5 che instrada attraverso Tor funziona. Puoi anche configurare proxy SOCKS4, SOCKS5, HTTP o HTTPS con autenticazione opzionale nelle impostazioni dell'app.

Perché SOCKS4 non protegge completamente la mia privacy come SOCKS5 e gli altri?

SOCKS4 non supporta la risoluzione DNS remota. Quando si usa Forza proxy personalizzato con un proxy SOCKS4, le query DNS vengono risolte localmente attraverso il DNS del tuo sistema/ISP, il che può esporre i domini a cui ti stai connettendo. SOCKS5 risolve il DNS attraverso il proxy stesso, prevenendo queste perdite. Se la privacy DNS è importante per te, usa SOCKS5, HTTP o HTTPS invece di SOCKS4 per il tuo proxy personalizzato. L'app mostrerà un avviso quando selezioni SOCKS4 come promemoria.

Come posso inviare stazioni darknet?

Invia stazioni .i2p e .onion tramite l'API Radio Registry. Le stazioni clearnet vanno nel database RadioBrowser.

Installa

Scarica l'ultimo APK qui o usa Obtainium per gli aggiornamenti automatici.

Verifica

Le release sono firmate con Sigstore. Per verificare, scarica sia il file .apk che il file .apk.sigstore dalla stessa release, poi esegui dalla directory contenente i file scaricati:

cd ~/Downloads
cosign verify-blob \
  --bundle deutsia-radio-v1.6.5.apk.sigstore \
  --certificate-identity siewargottlichschon@gmail.com \
  --certificate-oidc-issuer https://github.com/login/oauth \
  deutsia-radio-v1.6.5.apk

Compila

git clone https://github.com/deutsia/deutsia-radio.git
cd deutsia-radio
./gradlew assembleDebug

Richiede Java 21 LTS. Imposta JAVA_HOME=/usr/lib/jvm/java-21-openjdk prima di compilare.
